# Phần mềm cho thuê xe ô tô VF3

Bản demo website quản lý cho thuê xe ô tô bằng React + Vite.

## Chạy thử trên máy tính

```bash
npm install
npm run dev
```

## Đưa lên Vercel

1. Upload toàn bộ source code lên GitHub.
2. Vào Vercel → Add New Project.
3. Chọn repository này.
4. Bấm Deploy.

## Ghi chú

Bản này là demo giao diện, dữ liệu đang lưu tạm trong trình duyệt. Khi nâng cấp thật cần thêm database như Supabase hoặc Firebase.
